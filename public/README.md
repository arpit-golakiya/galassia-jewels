# Assets

Drop your media here:

- `logo.png` — Galassia Jewels logo (rendered at top of page, max ~80px tall)
- `product/1.jpg` … `product/4.jpg` — product gallery images (square works best, e.g. 1200×1200)
- `product/video.mp4` — optional product video (to be wired in later)

If files are missing the page falls back to remote placeholder images so layout always renders.
